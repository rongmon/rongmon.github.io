Astrophysical Codes Link to heading

We have developed a suite of extensive python-based software packages for astrophysical research. This page will be periodically updated. Note, all these codes are open source and available on GitHub. These are actively under development, so please check the respective GitHub repositories for the latest updates, documentation, and installation instructions. If you use any of these codes in your research, please consider citing the relevant zenodo release or the associated research papers.

rbcodes

A comprehensive suite of Python-based absorption line analysis pipeline and GUIs for studying the circumgalactic medium. Also includes, tools and GUIs for measuring galaxy properties.

rbvfit

A sophisticated Bayesian MCMC Voigt Profile fitting toolbox for analyzing absorption line systems in astrophysical spectra.

kcwitools

Python-based tools for Keck/KCWI integral field unit (IFU) data reduction and spectroscopic analysis.

musetools

Python-based tools for VLT/MUSE integral field unit (IFU) data reduction and analysis of spatially resolved spectroscopy.

iQNet v1

A deep learning model for QSO continuum fitting. Based on Liu & Bordoloi 2021 research on machine learning approaches to quasar spectral analysis.
